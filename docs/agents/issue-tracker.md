# Issue Tracker

This repo uses **GitHub Issues** for tracking work.

- Repository: `blessed1sagar/penna-ai` (private)
- CLI: `gh` (GitHub CLI)

## Commands

- Create: `gh issue create --title "..." --body "..." --label "..."`
- List: `gh issue list`
- Close: `gh issue close <number>`
- Add label: `gh issue edit <number> --add-label "..."`
- Remove label: `gh issue edit <number> --remove-label "..."`

Skills that use this: `to-issues`, `triage`, `to-prd`, `qa`
